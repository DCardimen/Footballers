# GRIDIRON — Life of a Player

A football career-simulation game (Phaser + canvas).

## Develop

```bash
npm install
npm run dev      # Vite dev server with hot reload at http://localhost:5173
npm run build    # production build -> dist/
npm run shot     # screenshot the running game (headless) for quick validation
```

The game currently lives in `index.html`. Systems are extracted into
readable modules under `src/` as they are worked on.
